This page provides a history of releases in reverse chronological order.

 
DateReleaseMajor Enhancements
06-Sept-2006 4.0.906
  • Brand new version, which provides modularized build.xml files. Initial tool support includes: javac, jar, javadoc, junit, pmd, checkstyle, findbugs, sclc.  Tools omitted for now: Emma, JBlanket, DependencyFinder, and all Hackystat sensor targets. 
  • Requires Java 5, Ant 1.7.x.
11-Feb-2005 3.4.211
  • Refactored Hackystat sensors into new package 'StackyHack'. No Hackystat stuff here.
  • Ant call to DependencyFinder now included.
10/27/2004 3.3.1027
  • Added JUnit, LOCC, Build, and JBlanket sensors.
09/21/2004 3.2.921
  • Removed duplicate junit taskdef to remove warning.
  • Removed extraneous constructor(String) from unit test class.
  • Improved documentation. 
09/20/2004 3.2.920
  • Updated to Ant 1.6, removed JBlanket support to simplify build system for beginning of semester.
09/29/2003 3.1.929
  • Fixes to JBlanket coverage in Ant task.
09/24/2003 3.1.924
  • Added JBlanket coverage.
09/10/2003 3.0.910
  • Updated the package to conform to latest Ant version and new JOSSE conventions.
  • Deleted Debug class in preparation for conversion to logging class.
03/30/2002 2.0330
  • Made the Stack and ClearStack classes serializable to support StackMVCdb.
  • Added a toString() method to Stack to simplify debugging.
3/08/2002 2.0308
  • Deleted Assert.java from project. Not used.
  • Added subclass ClearStack with iterator() method. This subclass supports the display of stack contents in StackMVC better than the toArray() method, which required copying the entire stack.
2/11/2002 2.0211
  • Improved test cases to call assertSame rather than assertTrue.
  • Improved naming of test methods.
  • Improved build.xml to use ${java.class.path} in junit and javadoc tasks.
1/22/2002 2.0122
  • Restructured with improved internal file and directory structure.
11/25/2001 1.1125
  • Added taskdefs for LOCC and j2h and jar files to Ant1.3\lib
  • Updated build.xml with dist, j2h, and locc tasks.
10/15/2001 1.1015
  • Removed build.bat, added setenv.bat and Ant1.3.
9/5/2001 1.0905
  • Initial release to class.
